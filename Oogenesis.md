# Oogenesis

See [Meiosis](Meiosis).
Produces only 1 ovum.

Starts with 1 primary oocyte (diploid) which divides into secondary oocyte and a first polar body. Secondary oocyte then divides into a ootid (haploid). The polar bodies cannot be fertilised, only the ootid.

Part of [The Human Reproductive System](The%20Human%20Reproductive%20System.md).
